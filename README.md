Create a venv in the directory

''
python -m venv venv
''

Then activate the venv

''
## on windows 

source venv/Scripts/activate
``
or 

''
## on mac or linux 

source venv/bin/activate 
''

install the requirements

''
pip install -r requirements.txt
''

run the bot

''
python discord_gemini_bot.py
''  

have fun!
